# GitHub / Community Announcement Templates

This folder contains ready-to-use release and community announcement templates for different channels.

## Files

- `release-github.md` — long-form GitHub Release description
- `announcement-slack.md` — short Slack announcement
- `announcement-discord.md` — Discord-friendly format
- `announcement-linkedin.md` — professional social post
- `announcement-x.md` — short X/Twitter post
- `contributor-onboarding-post.md` — onboarding automation announcement

## How to use

1. Copy the template matching your channel.
2. Replace placeholders:
   - `{{VERSION}}`
   - `{{DATE}}`
   - `{{HIGHLIGHTS}}`
   - `{{LINK_RELEASE}}`
   - `{{LINK_ISSUES}}`
3. Keep language consistent and link to canonical release notes.
